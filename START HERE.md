# Welcome to Vexidus

Vexidus is the ultimate L1 blockchain combining open-source and patent-protected technology. Please read this document before interacting with the blockchain tools provided in this repository.

## Folders

| # | Folder | Description |
|---|--------|-------------|
| 1 | **[Vexidus Validator](Vexidus%20Validator/)** | Everything you need to run a validator node and earn VXS rewards |
| 2 | **[Vexidus Smart Contracts](Vexidus%20Smart%20Contracts/)** | VSC token standards — create tokens, NFTs, and multi-token collections |
| 3 | **[IntentVM](IntentVM/)** | IntentVM transaction engine (patent-pending) |
| 4 | **[Vexidus Smart Wallets](Vexidus%20Smart%20Wallets/)** | VSA v2 wallet SDK for Ed25519 key management |
| 5 | **[Builder Toolkit](Builder%20Toolkit/)** | Developer tools and utilities |

## Getting Started

1. **Run a validator** — Start with the [Validator Setup Guide](Vexidus%20Validator/SETUP_GUIDE.md)
2. **Create a token** — Visit [VexForge](https://vexforge.xyz) or see [Smart Contracts](Vexidus%20Smart%20Contracts/)
3. **Read the docs** — Full documentation at [docs.vexidus.io](https://docs.vexidus.io)
4. **Get testnet VXS** — Visit the [faucet](https://vexswap.xyz)
